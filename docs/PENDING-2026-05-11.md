## ⏳ Pendente — próxima sessão

### P3D — ArgoCD Sync do llm-gateway

A Application `llm-gateway` pode ainda exibir `InvalidSpecError` até o ArgoCD controller reavaliá-la.

**Verificar antes de prosseguir:**
```bash
kubectl --kubeconfig ~/.kube/config-rbx describe application llm-gateway -n argocd | grep -A3 "InvalidSpecError"
```

Se o erro persistir após ~3 minutos, forçar hard-refresh (não é sync):
```bash
kubectl --kubeconfig ~/.kube/config-rbx annotate application llm-gateway -n argocd argocd.argoproj.io/refresh=hard
```

**Somente após o `InvalidSpecError` desaparecer**, autorizar sync:
```bash
argocd app sync llm-gateway
# ou via kubectl patch se CLI não disponível
```
